# 🔐 User Management Service - Spring Boot


**📌 Overview**
==================
This module provides basic user management functionality using Spring Boot and Spring Data JPA. It focuses on handling user registration and password updates.

**⚙️ Features**
==================

✅ User Registration

✅ User Validation (based on username & password)

✅ Password Update

**🧩 Components**
====================
Service Layer (Service1) – Handles business logic

Repository (Userrepo) – Interacts with database

Entity (Users) – Represents user data

DTO (UsersDto) – Transfers user input data

**🔄 Functionality**
========================
Register User: Stores new user details in the database

Update Password: Verifies user and updates password

User Check: Validates existing users

**🧠 Key Concepts
**
====================

Dependency Injection

Service Layer Architecture

DTO Pattern

Spring Data JPA
